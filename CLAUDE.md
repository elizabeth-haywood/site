## Code Review Guidelines

When reviewing PRs, focus on:

### Accessibility
- Ensure semantic HTML elements are used appropriately
- Check for alt text on images
- Verify sufficient color contrast
- Confirm keyboard navigation works

### Performance
- Images should be optimized and appropriately sized
- Minimize render-blocking resources
- Keep CSS and JS minimal and efficient

### SEO & Meta
- Check for appropriate meta tags (title, description, viewport)
- Verify Open Graph tags for social sharing
- Ensure proper heading hierarchy (h1, h2, etc.)

### Cross-Browser Compatibility
- CSS should work across modern browsers
- Avoid vendor-specific features without fallbacks

### Code Quality
- HTML should be valid and well-structured
- CSS should be organized and avoid duplication
- Keep code readable and maintainable
